# Filio 产品卖点与功能提炼文档 v1.0

> **产品定位**：专为英国会计师事务所打造的自动化客户文件收集与 Xero 同步平台。

---

## 一、 核心价值主张 (Core Value Proposition)

Filio 旨在解决英国会计师在 **MTD (Making Tax Digital)** 强制实施背景下，面临的客户文件收集低效、合规性差、手动操作繁琐等核心痛点。

| 核心卖点 | 详细描述 | 客户获益 |
| :--- | :--- | :--- |
| **极致安全承诺** | **行业唯一**做到“不处理、不储存”客户文件。文件通过加密通道直达 Xero。 | 消除数据泄露风险，完美符合 UK GDPR & ICO 合规要求。 |
| **客户零门槛上传** | 客户无需注册账号、无需设置密码。点击专属链接，60秒内完成上传。 | 极大提升客户配合度，减少催收文件的时间成本。 |
| **全自动同步 Xero** | 原生 API 集成，文件自动命名并直接归档至 Xero 对应的联系人目录下。 | 彻底告别手动下载、重命名、再上传的繁琐流程。 |
| **MTD 合规加速器** | 针对每季度 4 次的申报频率优化，将季度文件收集变成自动化流程。 | 轻松应对 MTD 带来的 4 倍行政负担，提升事务所人效。 |

---

## 二、 核心功能矩阵 (Key Features)

### 1. 智能文件收集 (Smart Collection)
*   **专属上传门户**：为每位客户生成唯一的安全链接，支持手机拍照直传。
*   **Magic Email (魔法邮箱)**：客户只需发送邮件附件即可完成上传，无需打开任何网页。
*   **自动文件命名**：根据“日期-客户名-文件类型”规则自动重命名（如：`2026-03-19-JohnSmith-Receipt.jpg`）。
*   **多格式支持**：支持 PDF、JPG、PNG、Excel、Word 等所有常用财务文件格式。

### 2. Xero 深度集成 (Xero Integration)
*   **一键 OAuth 授权**：30秒内完成 Xero 账户连接。
*   **联系人自动同步**：自动从 Xero 导入客户列表，无需手动录入。
*   **Files API 原生对接**：文件直接进入 Xero Files，支持实时状态更新。

### 3. 安全与合规 (Security & Compliance)
*   **透明管道架构**：Filio 仅作为加密传输中转站，不留存任何业务数据。
*   **银行级加密**：采用 AES-256-GCM 加密标准。
*   **本地化存储**：数据中心位于 AWS 伦敦，严格遵守英国数据保护法。
*   **资质认证**：已获得 ICO 注册认证及 Xero 官方集成认证。

### 4. 管理与协作 (Management)
*   **实时仪表板**：一目了然地追踪所有客户的上传进度。
*   **团队成员管理**：支持多名员工协作处理不同客户的文件请求。
*   **审计日志**：记录所有文件传输活动，确保操作可追溯。

## 三、 竞品差异化分析 (Competitive Edge)

相比于 **ContentSnare** 或 **手动流程**，Filio 的优势在于：

1.  **流程更短**：ContentSnare 需要通过 Zapier 才能连接 Xero，Filio 是原生 API 直连。
2.  **更安全**：ContentSnare 会在平台存储文件，Filio 坚持“零存储”原则。
3.  **成本更低且透明**：
    *   **ContentSnare**：按“请求数”计费，客户多时成本激增。
    *   **Filio**：按事务所固定收费，**无限客户、无限文件**，真正无上限。
