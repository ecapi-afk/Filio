# Upload Page — File Type & Category Style Guide

## Overview

The upload page (`app/portal/upload/portal-upload-client.tsx`) displays two parallel classification systems for uploaded files:

1. **File Type Badge** — Auto-detected from file extension (read-only display)
2. **Category Badge** — User-selected document type via dropdown (editable for `idle` files)

---

## 1. File Type Badges (Auto-detected from Extension)

These are display-only badges shown to the left of the filename. The color/icon is determined by the file extension.

### Badge Colors & Icons

| File Type | Extensions | Badge Color | Icon |
|---|---|---|---|
| Image | jpg, jpeg, png, heic, gif, webp, avif, bmp, tif, tiff | `bg-blue-100 text-blue-700 border-blue-200` | `<Image className="h-5 w-5 text-blue-500" />` |
| PDF | pdf | `bg-red-100 text-red-700 border-red-200` | `<FileText className="h-5 w-5 text-red-500" />` |
| Document | doc, docx, odt, rtf, txt | `bg-indigo-100 text-indigo-700 border-indigo-200` | `<FileIcon className="h-5 w-5 text-indigo-500" />` |
| Spreadsheet | xls, xlsx, csv, ods | `bg-green-100 text-green-700 border-green-200` | `<FileIcon className="h-5 w-5 text-green-500" />` |
| Presentation | ppt, pptx | `bg-orange-100 text-orange-700 border-orange-200` | `<FileIcon className="h-5 w-5 text-orange-500" />` |
| Archive | zip, rar, 7z | `bg-yellow-100 text-yellow-700 border-yellow-200` | `<FileIcon className="h-5 w-5 text-yellow-500" />` |
| Email | msg, eml | `bg-sky-100 text-sky-700 border-sky-200` | `<FileIcon className="h-5 w-5 text-sky-500" />` |
| File (default) | any other | `bg-gray-100 text-gray-600 border-gray-200` | `<FileIcon className="h-5 w-5 text-muted-foreground" />` |

### Label Text (getFileTypeLabel)

| File Type | Label | Logic |
|---|---|---|
| Image | `Image` | Extension in `['jpg', 'jpeg', 'png', 'heic', 'gif', 'webp', 'avif', 'bmp', 'tif', 'tiff']` |
| PDF | `PDF` | `ext === 'pdf'` |
| Document | `Document` | Extension in `['doc', 'docx', 'odt', 'rtf', 'txt']` |
| Spreadsheet | `Spreadsheet` | Extension in `['xls', 'xlsx', 'csv', 'ods']` |
| Presentation | `Presentation` | Extension in `['ppt', 'pptx']` |
| Archive | `Archive` | Extension in `['zip', 'rar', '7z']` |
| Email | `Email` | Extension in `['msg', 'eml']` |
| File | `File` | fallback |

## 4. Extension Badge (Left Icon Square)

The file card's left side shows a `w-10 h-10` rounded-lg square containing the **uppercase file extension** text (e.g. `PDF`, `DOC`, `XLS`). It is **not** an icon — it is the raw extension string rendered as bold white text.

| State | Background | Text Color |
|---|---|---|
| Normal (`idle`, `uploading`) | `bg-emerald-900` | `text-white` |
| Error (`error`) | `bg-red-900/50` | `text-white` |
| Done (`done`) | `bg-emerald-900` | `text-white` |

Generated by: `getExt(f.file.name)` — returns `name.split('.').pop()?.toUpperCase() || 'FILE'`

Example output in the square: `PDF` | `DOCX` | `XLS` | `PNG`

## 6. Allowed File Types (Accept Attribute)

```
.jpg, .jpeg, .png, .gif, .tif, .tiff, .bmp, .heic, .webp,
.pdf, .doc, .docx, .txt, .rtf,
.xls, .xlsx, .csv,
.ppt, .pptx,
.zip, .msg, .eml
```

**Max file size:** 10MB

## 9. Component Source Location

```
app/portal/upload/portal-upload-client.tsx
```

Key functions:

| Function | Purpose |
|---|---|
| `getExt(name)` | Extracts uppercase extension (e.g. `PDF`, `DOCX`) for the left icon square |
| `getFileTypeLabel(name)` | Returns display label (Image, PDF, Document, etc.) for the File Type Badge |
| `getFileTypeBadgeColor(name)` | Returns Tailwind classes for the File Type Badge |
| `getCategoryBadgeColor(category)` | Returns Tailwind classes for the Category Badge |
| `getFileIcon(type)` | Returns Lucide icon component for the file row |
| `formatSize(bytes)` | Formats bytes as KB or MB string |
